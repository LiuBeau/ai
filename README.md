# 🧠 LifeMate — 个人生活智能助手

> 一个从零构建的、具备长期记忆、自主规划与知识检索能力的桌面 AI Agent

LifeMate 是一个运行在 Windows 桌面上的个人生活助手，它不仅仅是一个“聊天机器人”，而是真正具备**自主思考、工具调用、记忆保持和自我反思**能力的智能体（Agent）。

## ✨ 核心特性

- 🧠 **自主规划**：基于 ReAct 模式，Agent 能自主拆解任务、调用工具、观察结果并迭代执行。
- 📝 **长期记忆**：通过 SQLite + ChromaDB 构建两层记忆系统，既记住你的偏好，也记住你的生活碎片。
- 🔧 **工具生态**：天气查询、日程管理、桌面通知、知识检索，并支持灵活扩展。
- 💬 **桌面交互**：基于 CustomTkinter 构建的现代化聊天界面，常驻桌面，随时响应。
- ⏰ **主动服务**：支持定时推送，每天早上自动推送天气与生活建议。
- 🧩 **RAG 知识库**：内置生活常识向量库，支持“穿衣搭配”、“旅游出行”等生活话题的语义检索。
- 🌀 **自我反思**：Agent 能识别失败调用并主动纠错，避免无效循环。
